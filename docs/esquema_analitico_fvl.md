# Esquema analítico FVL

## Definición general

La tabla analítica FVL contiene una fila por visita clínica con espirometría realizada.

La clave funcional de la tabla es `pat_visit_id`.

La regla funcional de inclusión es:

```sql
pv.FVLTest = 1
AND EXISTS (
    SELECT 1
    FROM dbo.FVLData AS fv_effort
    WHERE fv_effort.PatVisitID = pv.PatVisitID
      AND fv_effort.EffortTypeID = 0
)
```

Los resultados clínicos representativos se toman directamente de las filas generadas por Breeze:

- `EffortTypeID = 2`: resultado pre o basal.
- `EffortTypeID = 3`: resultado post broncodilatador.

Las maniobras individuales no se usan para reconstruir el resultado representativo.
